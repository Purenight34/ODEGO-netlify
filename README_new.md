# vibe-coding

이 프로젝트는 Vue 기반의 SPA 구조를 위한 예시 프로젝트입니다.

## 프로젝트 구조

- src/App.vue: 애플리케이션의 루트 컴포넌트입니다.
- src/main.js: Vue 앱을 생성하고 화면에 마운트합니다.
- src/style.css: 전역 스타일을 정의합니다.
- src/assets: 이미지와 스타일 파일을 저장합니다.
- src/components: 화면을 구성하는 재사용 가능한 컴포넌트들을 모아둡니다.
- src/data: 부산 관련 JSON 데이터 파일을 보관합니다.
- src/router: 라우팅 설정을 담당합니다.
- src/views: 페이지 단위의 뷰 컴포넌트를 담습니다.

## 실행 방법

1. 의존성 설치
   ```bash
   npm install
   ```
2. 개발 서버 실행
   ```bash
   npm run dev
   ```

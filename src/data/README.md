# data 폴더

이 폴더에는 프로젝트에서 사용하는 정적 데이터 파일을 보관합니다.

## 포함된 파일

- 부산_관광지.json: 부산 관광지 데이터를 담고 있습니다.
- 부산_레포츠.json: 부산 레포츠 데이터를 담고 있습니다.
- 부산_문화시설.json: 부산 문화시설 데이터를 담고 있습니다.
- 부산_쇼핑.json: 부산 쇼핑 데이터를 담고 있습니다.
- 부산_숙박.json: 부산 숙박 데이터를 담고 있습니다.
- 부산_여행코스.json: 부산 여행 코스 데이터를 담고 있습니다.
- 부산_축제공연행사.json: 부산 축제 및 공연 행사 데이터를 담고 있습니다.
